# **Basic Info**

this is a Linux cursor theme revolved around Seth from Deltarune, the originial cursor theme was made by [vev13417](https://x.com/vev13417) so this repository is just a port technically

# **Installation**

download the .zip file from the [releases page](https://github.com/yboyg41/seth-cur-4-linux/releases) and extract in either:
- ``/usr/share/icons/`` for all users
- ``~/.icons`` for a single user

and then just apply it from your DE's theming settings

# **"Building"**

this relies on ``xcursorgen`` for "building the xcursors, run this command to build the xcursor:

``xcursorgen <name>.cursor <standardname>``
 with <name> being the actual name of the .cursor file, and <standardname> being the name of the outputted xcursor file
